Abstract: Summary: Packed with examples, Simply SQL is a step-by-step introduction to learning SQL. You'll discover how easy it is to use SQL to interact with best-practice, robust databases. Rather than bore you with theory, it focuses on the practical use of SQL with common databases and uses plenty of diagrams, easy-to-read text, and examples to help make learning SQL easy and fun. Step through the basic SQL syntax Learn how to use best practices in database design Master advanced syntax like inner joins, groups, and subqueries Understand the SQL datatypes And much more ...
